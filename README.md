# Guess the Number

Простая консольная игра на Java.

## Описание

Компьютер случайным образом загадывает число от **1 до 100**, а задача игрока — угадать его.

После каждой попытки программа сообщает, больше или меньше загаданное число.

## Возможности

* Генерация случайного числа от 1 до 100.
* Проверка корректности пользовательского ввода.
* Обработка некорректных данных без завершения программы.
* Подсказки после каждой попытки.
* Подсчет количества попыток в текущей игре.
* Отображение лучшего результата среди всех сыгранных партий.
* Команда `RESULT` для просмотра статистики в любой момент игры.
* Возможность начать новую игру после победы.

## Используемые технологии

* Java
* IntelliJ IDEA

## Запуск

1. Клонировать репозиторий.
2. Открыть проект в IntelliJ IDEA.
3. Запустить класс `Main`.

## Команды

| Команда  | Описание                                  |
| -------- | ----------------------------------------- |
| `1-100`  | Попытка угадать число                     |
| `RESULT` | Показать статистику текущей и лучшей игры |

## Структура проекта

```text
src
├── Main.java
└── GameContextHandler.java
```

* **Main** — игровой цикл, обработка пользовательского ввода и логика взаимодействия с игроком.
* **GameContextHandler** — хранение состояния игры, генерация случайного числа и ведение статистики.
